# scenarios

This folder contains realistic Microsoft 365 support scenarios for the Dental Lab environment.

Each scenario explains:

The user’s request or problem
The Microsoft 365 feature used
The steps taken to resolve it
How the solution was tested
The final result

These scenarios are designed to demonstrate practical help desk and Microsoft 365 administration skills.

## Example Scenarios

User is out of office
New employee onboarding
Employee offboarding
Shared mailbox access
Mailbox storage issues
SharePoint permission requests
Password resets
Multi-factor authentication issues

## purpose of this doc

The scenarios folder shows how those services are used to solve common business and user support requests.
