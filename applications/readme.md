# Microsoft 365 Applications

This folder contains documentation for the Microsoft 365 applications used in the Dental Lab environment.

The documents focus on how employees use each application and how common support issues can be troubleshooted.

Administrative setup, permissions, and organization-wide configuration may be documented separately in the main docs folder.


## What Belongs in the Applications Folder

Documents in the applications folder should contain general and reusable information.

Examples include:

* How to access an application
* How to use common features
* General troubleshooting steps
* Common error messages
* User instructions
* Help desk troubleshooting checklists
* When an issue should be escalated

These documents should not normally focus on one specific employee or support ticket.

Example:

applications/teams/teams-troubleshooting.md

This document could explain general troubleshooting for:

* Teams sign-in problems
* Missing teams or channels
* Camera and microphone issues
* Notification problems
* Teams not opening
* Files not loading


## outlook

The Outlook folder contains documentation related to email, calendars, and shared mailbox access.

Possible topics include:

Accessing shared mailboxes
Sending from a shared mailbox
Automatic replies
Calendar troubleshooting
Outlook profile issues
Email synchronization problems

Shared mailbox creation and permission management are documented separately in the main shared mailbox document.

## OneDrive

The OneDrive folder contains documentation related to personal file storage and synchronization.

Possible topics include:

Signing in to OneDrive
Backing up Desktop, Documents, and Pictures
OneDrive sync problems
Files stuck on syncing
Missing files
Storage limits
Moving files to a replacement computer

OneDrive is mainly used for an individual employee's work files.

## Microsoft Teams

The Teams folder contains documentation related to communication, meetings, and collaboration.

Possible topics include:

Joining a team
Using channels
Starting or joining meetings
Camera and microphone troubleshooting
Teams sign-in issues
Missing teams or channels
Notification problems
Sharing files through Teams

Microsoft Teams may also connect to Microsoft 365 Groups and SharePoint.

## SharePoint

The SharePoint folder contains documentation related to shared department and organization files.

Possible topics include:

Accessing a SharePoint site
Uploading and downloading files
Creating folders
Sharing documents
File permission problems
Missing SharePoint sites
Opening SharePoint libraries through Teams
Synchronizing a SharePoint library with File Explorer

SharePoint is used for files that need to be shared between multiple employees or departments.

## Documentation Purpose

The purpose of this folder is to demonstrate common Microsoft 365 application support tasks.

Each application folder may contain:

A basic overview
User instructions
Troubleshooting steps
Screenshots from the lab
Example support scenarios

This folder will continue to grow as more Microsoft 365 application features and troubleshooting scenarios are tested in the lab.
