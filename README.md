# M365-Dental-Lab

This is a Microsoft 365 Business Premium lab that I created to practice real-world MSP-style administration tasks. The lab is based on a simulated dental office environment and is meant to help me improve my skills with Microsoft 365, Entra ID, Exchange, Intune, security settings, and documentation.

The goal of this project is not just to click around in the admin center, but to document the process like I would in a real IT support or MSP environment.

## Lab Scenario

For this lab, I created a fake dental office tenant called **Dental Lab** or **Meridian Dental** once the name change takes effect. The environment includes test users, Microsoft 365 Business Premium licensing, and admin configurations that would commonly be used for a small business client.

Example business scenario:

- Small dental office
- Multiple employees/users
- Microsoft 365 Business Premium licenses
- Email through Exchange Online
- Shared mailboxes for office workflows
- Groups for access management
- MFA and security policies
- Basic endpoint management with Intune


## Tools Used

- Microsoft 365 Admin Center

- Microsoft Intune Admin Center

## Lab Goals

The main goal of this lab is to practice and document common Microsoft 365 administration tasks, including:

- Creating and managing users
- Assigning Microsoft 365 licenses
- Creating groups
- Managing shared mailboxes
- Creating distribution lists
- User onboarding
- User offboarding
- Resetting passwords
- Managing MFA
- Configuring Conditional Access
- Managing Windows devices with Intune
- Creating basic security baselines
- Writing documentation like an MSP technician
